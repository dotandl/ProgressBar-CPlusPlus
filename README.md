# Progress Bar for C++
Simple CLI progress bar library for c++

## Build
To build make sure you have installed `CMake 3.8.0` or above and type:
```sh
$ mkdir build
$ cd build
$ cmake .. # You can specify the generator by typing: -G "Name of generator"
$ cmake --build .
```
